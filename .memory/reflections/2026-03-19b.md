**F-1 — SC-97 found and fixed a pre-existing circular import (not in ticket plan)**
`cli.py → bootstrap.py → db.py → ensure_schema → bootstrap._apply_lightweight_migrations`
caused `szimpla backfill-metadata` to crash at startup. The fix was clean (moved
`_apply_lightweight_migrations` into `db.py`) and within the ticket's spirit, but it wasn't
in the execution plan. Future parser-expansion tickets should add a verification step that
runs the CLI entrypoint before declaring AC complete — the test suite alone won't catch
import-layer bugs.

**F-2 — SC-98 requires a live backend for `gen:api`; autopilot cannot self-serve this step**
`npm run gen:api` hits `http://localhost:8000/openapi.json` at runtime. Autopilot runs
without a background server, so SC-98's S2 (frontend badge) will need either: (a) a manual
schema.d.ts update, or (b) a server startup within the delivery subagent. This should be
noted in the SC-98 plan before delivery is attempted.
